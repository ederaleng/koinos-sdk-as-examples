
# koinos-cdt-as-examples

  

## Installation

```sh

# with npm

npm install


# with yarn

yarn install

```

  

## Build examples

```sh

# with npm

npm run asbuild:calculator


# with yarn

yarn run asbuild:calculator

```

This will result in the generation of a contract.wasm file in the folder `calculator/build/release`and `calculator/build/debug`
